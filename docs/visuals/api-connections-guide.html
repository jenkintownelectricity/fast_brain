Authentication Types

Bearer Token

Standard OAuth2 bearer token authentication

Authorization: Bearer sk_live_xxx...
X-API-Key

API key in custom header (common for many services)

X-API-Key: your_api_key_here
Basic Auth

Username:password encoded in base64

Authorization: Basic dXNlcjpwYXNz...
None

Public APIs or internal services with IP allowlisting

No authentication header

Quick Start

Adding an API Connection

  1. Go to Skills Tab ‚Üí Outgoing API in the dashboard
  2. Click "Add Connection"
  3. Enter connection details:
    • Name: A friendly name (e.g., "Calendly Booking")
    • URL: Base URL of the API (e.g., "https://api.calendly.com")
    • Auth Type: Select authentication method
    • API Key: Your secret key (stored securely)
  4. Click "Save"
  5. Click "Test Connection" to verify it works

Example: Connecting to a Webhook
