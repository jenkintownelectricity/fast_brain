System Overview

Fast Brain LPU

  • Deployment Modal (Serverless)
  • System 1 (Fast) Groq + Llama 3.3 70B
  • System 2 (Deep) Claude 3.5 Sonnet
  • Target TTFB < 50ms
  • Region us-east

HIVE215 Voice Platform

  • Framework LiveKit Agents
  • STT Provider Deepgram (~30ms)
  • TTS Provider Cartesia (~30ms)
  • LLM Backend Fast Brain LPU
  • Default Skill tara-sales

Performance Targets

  • Total Voice Latency < 300ms
  • STT Latency ~30ms
  • LLM Latency (Fast) ~40-80ms
  • TTS Latency ~30ms
  • Tokens/sec > 400

API Reference

Method Endpoint Description
GET /health Health check - returns status, backend, skills_available, version
GET /v1/skills List all available skills with id, name, description, version
POST /v1/skills Create a new skill dynamically at runtime
GET /v1/skills/{skill_id} Get details for a specific skill
POST /v1/chat/completions OpenAI-compatible chat with skill parameter
POST /v1/chat/hybrid Hybrid System 1 + System 2 with filler phrases
POST /v1/chat/voice Hybrid + voice descriptions for TTS

We do all the technical work for you, so you get clarity without complexity. What kind of tools are you currently using to run your business?" } }], "skill_used": "tara-sales", "metrics": { "ttfb_ms": 42, "tokens_per_sec": 450 } }
