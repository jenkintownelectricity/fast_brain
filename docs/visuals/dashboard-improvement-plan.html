API API Endpoints (Ready for External Connections)
Skills API
GET /api/skills - List all skills
GET /api/skills/{id} - Get skill details
POST /api/create-skill - Create new skill
DEL /api/delete-skill/{id} - Delete skill
Golden Prompts API
GET /api/golden-prompts - List all prompts
GET /api/golden-prompts/{id} - Get prompt content
POST /api/golden-prompts/{id} - Save custom prompt
POST /api/golden-prompts/{id}/reset - Reset to default
GET /api/golden-prompts/export - Export all prompts
Fast Brain API
GET /api/fast-brain/health - Health check
POST /api/fast-brain/chat - Send chat request
GET /api/fast-brain/skills - List available skills
POST /api/fast-brain/skills - Create skill dynamically
Platform Connections API
GET /api/platforms - List all platforms
POST /api/platforms/{id}/connect - Connect to platform
POST /api/platforms/{id}/disconnect - Disconnect
POST /api/platforms/{id}/test - Test connection
