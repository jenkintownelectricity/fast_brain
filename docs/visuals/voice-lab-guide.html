Supported Providers

  • Works reliably from Modal
  • Great for testing

Expressive voice synthesis with emotion control

  • No API key required
  • 10+ emotion presets
  • Skill-specific voice personas
  • GPU-powered (30-60s cold start)

Professional voice cloning with instant voice cloning

  • High-quality voice cloning
  • Instant voice from 1 sample

  • Requires API key
Cartesia Premium

Ultra-low latency voice synthesis

  • Voice cloning support
  • Streaming audio
  • Requires API key

Quick Start Guide

Free Voice Testing (No API Key)

  1. Go to Voice Tab in the dashboard
  2. Click "Create Voice Project"
  3. Select "gTTS" or "Parler TTS" as provider
  4. Enter a name for your voice project
  5. Click "Create"
  6. Open project and click "Test Voice"
  7. Listen to audio - it works!

Premium Voice Cloning (ElevenLabs)

  1. Go to Settings → API Keys
  2. Enter your ElevenLabs API key
  3. Go to Voice Tab → Create Voice Project
  4. Select "ElevenLabs" as provider
  5. Upload audio samples (at least 30 seconds total)
  6. Add transcripts for each sample
  7. Click "Train Voice"
  8. Wait for training (usually 1-5 minutes)
  9. Test your cloned voice!

Troubleshooting

Audio Not Playing (0:00 / 0:00 duration)

  1. Check browser console for errors (F12 → Console)
  2. Verify audio file was generated (check for error message)
  3. Try a different text or shorter phrase
  4. Ensure gTTS is working (test with free provider first)
  5. Check Modal logs for server-side errors

Training Failed (Premium Providers)

  1. Verify API key is valid in Settings → API Keys
  2. Ensure audio samples are clear (no background noise)
  3. Check sample length (minimum 30 seconds total)
  4. Verify audio format is supported (MP3, WAV, M4A)
  5. Check provider's dashboard for quota limits
